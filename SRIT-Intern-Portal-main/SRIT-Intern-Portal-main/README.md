# Intern Portal – SRIT Internship Hub

-- Project Overview
Intern Portal is a backend-driven web application developed to help students explore and apply for verified internships easily. The platform provides a secure and organized system where students can register, log in, and apply for internships based on their branch and eligibility criteria.

## Demo Video:
https://drive.google.com/file/d/1DpLvzuGhglYkaKCwdrEETQvT9yxaV43Y/view?usp=sharing

The application supports role-based access control with Admin and Student roles.

-- Features
- Secure User Registration & Login
- Role-Based Access Control (Admin & Student)
- Branch-wise Internship Filtering
- Internship Application Tracking
- Admin Dashboard to Manage Internships
- RESTful API Architecture
- Real-Time Database Integration

---

-- Technologies Used
- Backend: Node.js, Express.js  
- Database: MongoDB  
- Authentication: JWT (JSON Web Token)  
- API Testing: Postman  
- Version Control: Git & GitHub  

---

-- User Roles

🔹 Admin
- Add new internship opportunities
- Update internship details
- Delete internship listings
- View student applications

🔹 Student
- Register and Login
- Browse internships
- Filter internships by branch
- Apply for internships
- Track application status

---

-- Project Architecture
- RESTful APIs built using Express.js
- MongoDB used for storing users, internships, and applications
- JWT-based authentication for secure access
- Middleware used for role-based authorization
